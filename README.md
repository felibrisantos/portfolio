A high-end, editorial-inspired portfolio for **Felipe Brigagão**, Fullstack & AI Architect. This project showcases a deep integration between modern web development and advanced Artificial Intelligence architectures.

## ⚡ Quick Start

```bash
# Clone the repository
git clone https://github.com/felibrisantos/portfolio.git

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## 🎨 Design Philosophy

- **Intentional Minimalism:** Every element serves a specific purpose, rejecting generic templates for a bespoke, editorial aesthetic.
- **Micro-interactions:** Powered by **Framer Motion** for fluid, high-performance transitions and staggered reveals.
- **Typography-led:** Featuring a sophisticated pairing of **Bodoni** (Luxury/Editorial) and **Sora** (Modern Sans-serif).
- **Interactive Feedback:** Real-time hover states and custom UI components like the `BentoCard` and `TerminalPreview`.

## 🛠️ Tech Stack

### Frontend & UI
- **React 19:** Utilizing the latest features for state management and rendering.
- **Next.js 15 (App Router):** Optimized routing and SEO.
- **Tailwind CSS v4:** Modern, utility-first styling with high-performance CSS.
- **Framer Motion:** Advanced declarative animations.
- **Lucide React:** Minimalist iconography.

### Intelligence & AI
- **GenAI Architectures:** Integration with OpenAI, Anthropic, and Google Gemini.
- **MCP (Model Context Protocol):** Standardized context sharing for AI agents.
- **RAG & Vector DBs:** Knowledge-augmented generation patterns.

### Backend & Infrastructure
- **Node.js & Prisma:** Robust data orchestration and type-safe ORM.
- **Vercel:** Optimized deployment and Edge functions.

## 🚀 Key Projects

- **CYMA Architect Agent:** Multi-agent orchestration for real-time environment redesign.
- **CYMATRIX:** High-performance control center for digital operations.
- **CYMADISPLAY:** Production-ready content management platform.
- **Portal TM-HNK:** Enterprise-scale operations hub for Heineken Brasil.
